This app can be used to see how a web page is displayed indifferent devices like phone,tab,desktop.
Start app by double clicking run.bat
Put the html file under test in the same location where app.js is present.
Now if the file name is home.html. Open http://localhost:3003/index.html and enter http://localhost:3003/home.html
in input text field and hit enter.
Different viiews of home.html will be displayed.
